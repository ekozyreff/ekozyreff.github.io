### Connections between Integer Programming and Neural Networks

In this project I implemented the model described in the paper "[Deep neural networks and mixed integer linear optimization](https://doi.org/10.1007/s10601-018-9285-6)", by M. Fischetti and J. Jo. [This post](../posts/2021/05/21/how-to-fool-a-neural-network.html) explains the project in more detail. The code is available [here](https://github.com/ekozyreff/fischetti_jo_2018).
